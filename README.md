# Power BI Churn Dashboard

This project presents an interactive Power BI dashboard for customer churn analysis.
Machine learning (Logistic Regression) is used to predict churn probability.
The model outputs are integrated into the dashboard for insights.
Key KPIs, trends, and customer behavior patterns are visualized.
Data preprocessing and feature analysis were performed before modeling.
The dashboard focuses on clear, actionable business insights.
This project was completed as part of an internship task.
